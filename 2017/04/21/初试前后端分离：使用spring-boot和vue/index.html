初试前后端分离：使用spring boot和vue

发表于 2017-04-21

前后端分离指前后端只通过 JSON 来交流，组件化、工程化不需要依赖后端去实现。本文将选择 spring boot 和 vue（使用vue-cli构建）进行实现。在阅读此篇文章之前希望你已经简单了解过 spring boot 和 vue。

1.创建 spring boot 项目

你可以去自行去官网创建并下载，或者使用你所熟悉的IDE创建，本项目只需要勾选Web依赖。

写一个简单的 controller

在 spring boot 的 application 类的同级目录下创建 controller 包，并在其中创建 HomeController 类，该类用于处理一些 rest 请求，代码如下:

@RestController
public class HomeController {
    @RequestMapping(value = "/", method = RequestMethod.GET)
    public String home() {
        return "Hello World!";
    }
}

后端用 cors 来解决跨域问题

跨域问题是前后端分离势必要解决的问题之一，本人选择了使用 cors。

cors 是一个 W3C 标准，全称是”跨域资源共享”（Cross-origin resource sharing）。它允许浏览器向跨源服务器，发出 XMLHttpRequest 请求，从而克服了AJAX只能同源使用的限制。关于具体的 cors 工作方式请移步阮一峰老师的详解。

在 spring boot 的 application 类的同级目录下创建 config 包，并在其中创建 CorsConfig 类，该类写了一些 cors 的配置，代码如下:

@Configuration
public class CorsConfig {
    @Bean
    public CorsFilter corsFilter() {
        UrlBasedCorsConfigurationSource source = new UrlBasedCorsConfigurationSource();
        CorsConfiguration corsConfiguration = new CorsConfiguration();
        corsConfiguration.setAllowCredentials(true);
        corsConfiguration.addAllowedOrigin("*");
        corsConfiguration.addAllowedHeader("*");
        corsConfiguration.addAllowedMethod("*");
        source.registerCorsConfiguration("/**", corsConfiguration);
        return new CorsFilter(source);
    }
}

到此后端的任务全部结束。

创建前端 vue 项目

首先是用 vue-cli 创建一个 vue 项目。（如果不知道什么是 vue 赶紧移步这里呀）

使用axios来解决跨域问题

axios 是基于 Promise 的 HTTP 请求客户端。原本 vue 有个叫 vue-resource 的插件去实现跨域，但后来作者放弃了更新，而是推荐使用 axios 。

首先在项目中引入 axios :

$ npm i axios --save

在 src 包下的 main.js 中加入一些 axios 的配置，代码如下：

import Vue from 'vue'
import App from './App'
import axios from 'axios'

Vue.config.productionTip = false

axios.defaults.headers.post['Content-Type'] = 'application/x-www-form-urlencoded'
/* baseURL是 spring boot 运行的端口 */
axios.defaults.baseURL = 'http://localhost:8080'

Vue.prototype.$http = axios

new Vue({
  el: '#app',
  template: '<App/>',
  components: { App }
})

修改 config 包下的 index.js 把端口改成3000：

dev: {
  env: require('./dev.env'),
  port: 3000,
  autoOpenBrowser: true,
  assetsSubDirectory: 'static',
  assetsPublicPath: '/',
  proxyTable: {},
  cssSourceMap: false
}

修改 src 包下的 App.vue :

<template>
  <div id="app">
    <img src="./assets/logo.png">
    <h2>{{msg}}</h2>
  </div>
</template>

<script>
  export default {
    data () {
      return {
        msg: ''
      }
    },
    created () {
      this.$http.get('/home')
        .then((response) => {
          this.msg = response.data
        })
        .catch((error) => {
          console.log(error)
        })
    }
  }
</script>

created 钩子当事件初始化后被调用，功能是通过 axios 发送一个 get 请求，如果请求成功把返回的 json 对象赋值给 msg 。

好的，到这里前端的东西也差不多写完了，现在可以跑起来看一下效果了。(前端启动的时候记得先跑一下 npm install)

可以看到后端返回的“Hello World!”已经被渲染在页面上了。
